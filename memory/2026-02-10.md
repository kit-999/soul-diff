### Technical Setup
- **Storage:** Vercel Blob (not KV) — each app has its own blob store
- **Code:** `/Users/jerika/.openclaw/workspace/public/therapy-tracker/` and `/therapy-worksheets/`
- **PDFs:** `/Users/jerika/.openclaw/workspace/therapy files/` (converted to PNG for vision reading)
- **API:** `/api/sync` uses `@vercel/blob` with `list()` to find correct blob URL

### Key debugging lesson
Initial blob API wasn't working because I hardcoded the wrong URL. Had to use `list()` to find the actual blob URL dynamically. The blob URLs look like `https://7keuenxytwib8vtw.public.blob.vercel-storage.com/filename.json` — not predictable from the store name.
